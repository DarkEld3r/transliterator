## Overview

Simple translit convertor.

## Building

### Instal FreeType
Conrod uses [FreeType](http://www.freetype.org/) for font rendering.

### Build
```sh
git clone https://github.com/DarkEld3r/transliterator
cd transliterator
cargo build
```
## Running

```sh
cargo run --bin transliterator
```
